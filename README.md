# COMP3161_GROUP_PROJECT
A backend REST API built with Flask and MySQL for a university course management 
system. Designed as part of a group project for COMP3161 (Introduction to Database 
Management Systems) at the University of the West Indies.

This repository covers the Course Management & Enrollment module, handling the 
creation of courses, student enrollment, lecturer assignment, and member retrieval.

Built with raw SQL queries (no ORM) as required by the project specification.
